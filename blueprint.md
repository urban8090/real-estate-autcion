# Project Blueprint

## Overview

This project is a simple web application that provides various fun and useful features. The initial feature is a lottery number generator.

## Current Plan: Lotto Number Generator

### Objective
Create a feature that allows users to generate a set of random lottery numbers.

### Steps
1.  **Modify `index.html`**:
    *   Update the title and heading to reflect the "Lotto Number Generator" feature.
    *   Create a container to display the generated numbers.
    *   Add a button to trigger the number generation.
2.  **Modify `style.css`**:
    *   Style the overall layout, making it visually appealing.
    *   Style the number container and the individual number balls.
    *   Style the "Generate" button.
3.  **Modify `main.js`**:
    *   Implement a function to generate 6 unique random numbers between 1 and 45.
    *   Add an event listener to the button that calls the number generation function.
    *   Display the generated numbers in the UI.
